A feature-rich scientific calculator built using Flask (Python) and JavaScript. It supports multiple calculation modes, including:

 Algebra Mode (Basic arithmetic, exponents, roots);
 Trigonometry Mode (sin, cos, tan, radians, degrees);
 Complex Numbers (real, imaginary, conjugates, arguments);
 Matrix Operations (determinant, inverse, dot product);
 Programming Mode (binary, hexadecimal, octal, bitwise operations);
 Statistics Mode (mean, median, standard deviation).




 Backend: Flask (Python) for handling calculations;
 Frontend: HTML, CSS, and JavaScript for a user-friendly UI;
 Responsive UI with animations and smooth mode transitions.
